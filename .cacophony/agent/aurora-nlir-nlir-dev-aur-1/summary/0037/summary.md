# Session summary — golf #24 (three-way zoom) + target #22 (capability question)

## What landed
- examples/golf-aur1-24-zoom.sh — `'<doc>';[#$,~$,>$]`: one doc, three altitudes from a single
  push (peeked 3x). #$ = the TAG/title (zoom out), ~$ = the ABSTRACT (mid), >$ = the full
  ELABORATION (zoom in). A circuit-breaker paragraph → "Circuit breaker" / one-line summary / a
  full explainer. Index-entry + abstract + deep-dive in one expression; distinct from review-kit
  (2 opposed lenses) and compare (2 candidates).
- examples/target-aur1-22-capability.sh — 13th `?` framing: 'you support webhooks'? (21c) → "Do
  you support webhooks?" A present-tense capability seed → the "Do you…?" frame (vs #18 past-tense
  "Did…?"). ? picks Do/Did from the seed's tense.

## Operator-takeaway
The stack lets one source be viewed at N depths without renaming it (# tag / ~ abstract / >
elaboration). And ? distinguishes present vs past ("Do you" vs "Did") from the seed's tense.
